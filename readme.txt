Interactive Rongali Bihu greeting card with name customization and social sharing features.

== Description ==

This plugin creates an interactive Rongali Bihu greeting card that allows users to:
- Enter their name to personalize the greeting
- Share the greeting on social media
- Experience festive animations and music

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rongali-bihu-greeting` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode `[rongali_bihu_greeting]` in any post or page

== Frequently Asked Questions ==

= How do I change the background image? =
Upload your image to the assets/images folder and update the CSS file.

= How do I change the music? =
Replace the bihu-music.mp3 file in the assets/audio folder.
